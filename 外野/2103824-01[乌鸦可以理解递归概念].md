多年来，科学家倾向于认为人类是唯一理解递归概念的动物。比如，在“猫追的老鼠跑了”（The rat the cat chased ran）的表达中，“猫追的”（the cat chased）这个句子被嵌入了另一个句子——一层套一层的中心嵌入，可以让句子无限进行下去。两年前有研究发现，一些种类的猴子能像3-4岁的人类儿童一样理解递归的概念。而如今，一项发表在《科学进展》的研究认为，乌鸦也拥有相似的认知能力。

研究者试图训练乌鸦理解递归序列。他们向乌鸦展示四个彩色括号，比如{ } [ ]，或者{ } ( )。而乌鸦需要学会把两对括号用中心嵌入的方式套在一起，比如{ [ ] }或者{ ( ) }。若乌鸦啄括号的顺序正确，它们就会获得奖励。而乌鸦掌握了简单的递归序列后，科学家又会创建更长的序列来训练乌鸦，例如{ [ ( ) ] }。研究者发现，乌鸦能抓住递归序列的规律，用正确的顺序啄括号，并且比一些猴子更加擅长，从两对括号增加到三对括号也没有降低乌鸦完成任务的速度。研究认为，递归能力并非人类或灵长类动物独有的能力。
