只是时代不需要了</strong></strong>在这封给员工的信中，大众汽车变速器（上海）分析了如今行业的未来发展趋势，以及自家公司产品不景气的原因，主要在于传统燃油车向新能源汽车的转型，让公司的未来变得扑朔迷离。<img src="https://img.36krcdn.com/20221118/v2_da2f9e524b604bf3a1a0662d40fcb2ac_img_000" id="aimg_R33Kj" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：大众近年来，我国自动变速器的市场份额增长速度较快，以每年约 5%的速度增长，在2020的汽车市场，自动变速器渗透率已超过70%。而手动挡变速器目前主要以商用车领域为主，除了职业司机和喜欢操纵感的少数人依旧使用手动挡之外，已经很少有人购买手动挡车型了。预计2023年以后，我国汽车行业的自动变速器渗透率将达到90%以上。所以在市场的压迫下，手动变速器的日子已经走到头了。大众汽车变速器全员信中也提到，由于生产线、生产规模和员工人数的不断缩减，手动变速器也只剩下MQ200这一款产品，曾经满大街的手动挡汽车现在变成了小众产品，因此公司在今年年初就已经做出了停产手动变速器的决定。这个工厂在2001年10月份成立，大众、一汽、上汽分别持股60%、20%和20%，总投资达到1.26亿美元，主要产品是MQ系列的手动变速器，在生产之初，服务于大众的A级车型。目前上汽大众旗下还提供手动挡的车型，有桑塔纳、朗逸、Polo和途铠，但在2023年，途观和帕萨特的下一代产品将是第一批不配置手动变速箱的车型。<img src="https://img.36krcdn.com/20221118/v2_b3d4b40bf953484199b4d87cbed1a51d_img_000" id="aimg_cQ6eF" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：大众根据计划，2030年，大众在欧洲、美国和中国销售的所有车型，都将只配备自动变速箱。而此次上海分公司大众汽车变速器公司宣布即将停产之后，就只剩下专注自动变速器制造的大连分公司和天津分公司还在营业。不仅是大众，在今年6月奔驰公开表示，随着汽车电气化程度的提高，客户需求也逐渐往电动汽车零部件、电池和电力驱动系统靠近，奔驰决定在2023年逐步停产并淘汰所有的手动挡车型。而在这之前，奥迪、保时捷、法拉利已经宣布停产手动挡车型，丰田也已经在一些车型上完全摒弃了手动挡，只生产自动挡。传统车企拥抱电气化时代的到来，而手动挡车型从以前的常见交通工具，变成了一车难求的小众物品。<img src="https://img.36krcdn.com/20221118/v2_63c8685d23234144993163978b0031cf_img_000" id="aimg_K1ODA" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：大众不过在大众停产手动变速器后，还需要考虑的问题是，手动变速器在后续的改进与维修是如何进行。这点大众变速器（上海）并没有给出确切的答案，但从这段时间到明年3月，公司进入关停清算流程，应该会给出一些具体解决方案。<strong>02 手动挡何去何从？</strong>现在新能源大潮下，从“看不懂“到”被抛弃“可能只需要很短的时间。在以前，省油和价格低是手动挡车型优势。但对于目前新能源汽车市场来说，手动变速箱的优势几乎完全不存在了。先来看几组数据，中国市场调查研究中心调查显示，2009年，手动挡（MT）在国内乘用车的占比达到了68%，但到了2021年，手动挡占比锐减至12%。<img src="https://img.36krcdn.com/20221118/v2_add8e530d6f74eb08947391478cec169_img_000" id="aimg_Nd9rg" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：中国市场调查研究中心对于大众品牌销量车型来说，手动挡新车的销量也在不断走低，2020年全年销售手动挡新车211.7万辆，市场份额有10.7%，2021年销量下滑27.9%，市场份额也下降至7.2%。到了2022年上半年，手动挡新车的市场份额仅剩下4.8%。<img src="https://img.36krcdn.com/20221118/v2_d7c4a04a97544ba6aa2a51c0d20e18ab_img_000" id="aimg_mYhY3" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：大众在如今新能源汽车的渗透率持续提升的情况下，这个市场占比大概率还会继续降低。从国内半壁江山到即将退出市场，这种衰落的速度，就已经说明了手动挡汽车已经跌落神坛。对于新能源车型来说，是通过电池向电动机提供电能，驱动电动机运转，从而推动汽车行驶。纯电动汽车核心是三电系统，在动力设计上直接跨越了变速箱这类零部件。再从电车的驱动结构来看，其电机在工作的时候，只需要通过控制电流的大小，就能掌控电机的转速范围和扭矩输出。这也意味着，新能源汽车自身的驱动，并不是靠变速箱进行控制的。所以不需要通过车辆变速箱进行物理的介入，去实现车辆变速。所以淘汰手动变速器有如汽油发动机淘汰蒸汽发动机一样，历史[车轮](https://36kr.com/project/1817557736851337)不断前进，技术在不断革新，适合时代的才是最正确的选择。当然，电气化的来临，并不意味着手动挡汽车会完全消失，在某种意义上来讲，手排挡是一些汽车中的”灵魂“所在。在未来可能会出现在一部分豪华品牌极少数车型上，变成“小众奢侈品”。配以更大功率的发动机，以此实现更强的操作和驾驶感。<strong>03 转型势不可挡</strong>其实大众在恰当的时候“断舍离”，也会更有利于未来的电动化发展。从资源配置的角度来说，大众停止提供手动变速箱，不把钱投在完全没有市场前景的配件上也是非常正确的。特别是大众的转型节奏一直较为激进的情况下，高昂的成本研发就成了大众电气化转型道路上的一座大山。特别是自今年以来，大众更是进一步加速转型，保时捷单独IPO，获得的资金收益除了49%作为股息支付给股东外，其余的资金则用于大众汽车集团的电气化转型。甚至大众汽车集团为了获取更多的资金正在评估兰博基尼IPO（首次公开募股）的可能性。<img src="https://img.36krcdn.com/20221118/v2_df5cb45a018b4e6982c900cdcabf657b_img_000" id="aimg_oewQ4" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)来源：大众另外，大众汽车集团官宣与中国智能计算平台[地平线](https://36kr.com/project/1678360559531017)公司合作共同成立合资公司，也是为了其电气化转型做铺垫。今年前三季度，大众集团共交付纯电动车型36.64万辆，其中欧洲市场达到21.19万辆。按计划，今年大众将会生产80万辆纯电动汽车，2023年要达到130万辆，并在2030年实现电动汽车的占比总销量一半。<img onmouseover="img_onmouseoverfunc(this)"/)来源：大众现在，大众直接关停已经成立二十多年之久的变速器工厂，而不是选择去逐渐转型到新能源工厂的做法，也能从一定程度上表达出大众对于电气化转型的决心。现在大众已经首先“封杀”手动挡变速箱，可以预见的是，其它车企也会跟进。以后你再想开手动档的燃油车，除了去二手车市场，可能得去赛道了。本文来自微信公众号[“超电实验室”（ID：SuperEV-Lab）](http://mp.weixin.qq.com/s?__biz=Mzg5MTc3NjgxNQ==&amp;mid=2247495042&amp;idx=1&amp;sn=410fdc161d6df59f8c6820e8bfd54ceb&amp;chksm=cfca97c2f8bd1ed4ccd1f1d3081faf584e521b64d7466395b54b9442dc1db903552d1059e641#rd)，作者：王磊
